Summary: Load stock ticker data for analysis, specifying the start and end dates, data source, and time interval. You can also choose to load weekly or monthly data, include pre/after market hours, load data from a custom file, and export the data. The command supports various data sources and can show performance information.

Examples:
- Load stock data for a specific symbol: stocks/bt/load -t <SYMBOL>
- Load stock data for a specific symbol with a custom start and end date: stocks/bt/load -t <SYMBOL> -s 2020-01-01 -e 2022-12-31
- Load intraday stock data with a 30-minute interval: stocks/bt/load -t <SYMBOL> -i 30
- Load stock data including pre/after market hours: stocks/bt/load -t <SYMBOL> -p
- Load stock data from a custom file: stocks/bt/load -t <SYMBOL> -f FILEPATH
- Load monthly stock data: stocks/bt/load -t <SYMBOL> -m
- Load weekly stock data: stocks/bt/load -t <SYMBOL> -w
- Load stock data and show performance information: stocks/bt/load -t <SYMBOL> --performance
- Load stock data and export it to a csv file: stocks/bt/load -t <SYMBOL> --export csv
- Load stock data and save it to a specific excel sheet: stocks/bt/load -t <SYMBOL> --sheet-name Sheet1
- Load stock data from a specific data source: stocks/bt/load -t <SYMBOL> --source AlphaVantage
