Summary: Load stock ticker data for analysis, including start and end dates, intraday intervals, pre/after market hours, and data source selection. You can also load custom files, monthly or weekly data, and export raw data in various formats. Performance information can be displayed as well.

Examples:
- Load <SYMBOL> with default settings: stocks/ins/load -t <SYMBOL>
- Load <SYMBOL> with a specific date range: stocks/ins/load -t <SYMBOL> -s 2020-01-01 -e 2021-12-31
- Load <SYMBOL> with intraday data at 15-minute intervals: stocks/ins/load -t <SYMBOL> -i 15
- Load <SYMBOL> including pre/after market hours: stocks/ins/load -t <SYMBOL> -p
- Load custom file for analysis: stocks/ins/load -f FILEPATH
- Load <SYMBOL> with monthly data: stocks/ins/load -t <SYMBOL> -m
- Load <SYMBOL> with weekly data: stocks/ins/load -t <SYMBOL> -w
- Load <SYMBOL> and show performance information: stocks/ins/load -t <SYMBOL> --performance
- Load <SYMBOL> and export raw data in xlsx format: stocks/ins/load -t <SYMBOL> --export xlsx
- Load <SYMBOL> from a specific data source: stocks/ins/load -t <SYMBOL> --source AlphaVantage
