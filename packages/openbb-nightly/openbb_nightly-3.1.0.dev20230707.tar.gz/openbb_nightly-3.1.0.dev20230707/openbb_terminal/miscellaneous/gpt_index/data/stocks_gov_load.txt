Summary: Load a stock ticker to perform analysis on with options to specify the date range, data interval, and data source. You can also load custom files, choose between monthly or weekly data, and display performance information. Data can be exported in various formats and saved to specific excel sheets.

Examples:
- Load a stock ticker for analysis: stocks/gov/load -t <SYMBOL>
- Load a stock ticker with a specific date range: stocks/gov/load -t <SYMBOL> -s 2021-01-01 -e 2021-12-31
- Load a stock ticker with intraday data at 15-minute intervals: stocks/gov/load -t <SYMBOL> -i 15
- Load a stock ticker and include pre/after market hours data: stocks/gov/load -t <SYMBOL> -p
- Load a stock ticker using a custom file: stocks/gov/load -t <SYMBOL> -f FILEPATH
- Load a stock ticker with monthly data: stocks/gov/load -t <SYMBOL> -m
- Load a stock ticker with weekly data: stocks/gov/load -t <SYMBOL> -w
- Load a stock ticker and display performance information: stocks/gov/load -t <SYMBOL> --performance
- Load a stock ticker with a specific data source: stocks/gov/load -t <SYMBOL> --source AlphaVantage
- Load a stock ticker and export data in xlsx format: stocks/gov/load -t <SYMBOL> --export xlsx --sheet-name Sheet1
