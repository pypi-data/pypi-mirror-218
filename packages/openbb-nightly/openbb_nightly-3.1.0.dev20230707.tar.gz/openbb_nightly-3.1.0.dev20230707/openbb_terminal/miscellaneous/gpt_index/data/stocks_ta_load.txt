Summary: Load stock ticker data for technical analysis, specifying the ticker, date range, interval, and data source. You can also choose to include pre/after market hours, load monthly or weekly data, and show performance information. Additionally, you can export the raw data and save it to a specific Excel sheet.

Examples:
- Load <SYMBOL> with default settings: stocks/ta/load -t <SYMBOL>
- Load <SYMBOL> with a custom date range: stocks/ta/load -t <SYMBOL> -s 2021-01-01 -e 2021-12-31
- Load <SYMBOL> with 30-minute intraday data: stocks/ta/load -t <SYMBOL> -i 30
- Load <SYMBOL> including pre/after market hours: stocks/ta/load -t <SYMBOL> -p
- Load <SYMBOL> using AlphaVantage as the data source: stocks/ta/load -t <SYMBOL> --source AlphaVantage
- Load <SYMBOL> with monthly data: stocks/ta/load -t <SYMBOL> -m
- Load <SYMBOL> with weekly data: stocks/ta/load -t <SYMBOL> -w
- Load <SYMBOL> and show performance information: stocks/ta/load -t <SYMBOL> --performance
- Load <SYMBOL> and export raw data in xlsx format: stocks/ta/load -t <SYMBOL> --export xlsx
- Load <SYMBOL> and save data to a specific Excel sheet: stocks/ta/load -t <SYMBOL> --sheet-name Sheet1
