Summary: Load stock ticker data for analysis, including start and end dates, interval, pre/after market hours, and data source. You can also load custom files, choose between monthly or weekly data, and export raw data into different formats. Performance information can also be displayed.

Examples:
- Load <SYMBOL> with default settings: stocks/ba/load -t <SYMBOL>
- Load <SYMBOL> with a specific start and end date: stocks/ba/load -t <SYMBOL> -s 2021-01-01 -e 2021-12-31
- Load <SYMBOL> with intraday data at 15-minute intervals: stocks/ba/load -t <SYMBOL> -i 15
- Load <SYMBOL> with pre/after market hours data: stocks/ba/load -t <SYMBOL> -p
- Load custom file for analysis: stocks/ba/load -f FILEPATH
- Load <SYMBOL> with monthly data: stocks/ba/load -t <SYMBOL> -m
- Load <SYMBOL> with weekly data: stocks/ba/load -t <SYMBOL> -w
- Load <SYMBOL> and show performance information: stocks/ba/load -t <SYMBOL> --performance
- Load <SYMBOL> and export raw data to a JSON file: stocks/ba/load -t <SYMBOL> --export json
- Load <SYMBOL> using a specific data source: stocks/ba/load -t <SYMBOL> --source AlphaVantage
