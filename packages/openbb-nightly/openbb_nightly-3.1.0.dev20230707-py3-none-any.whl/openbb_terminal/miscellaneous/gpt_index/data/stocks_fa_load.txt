Summary: Load a stock ticker for analysis, specifying the start and end dates, data source, and time interval. You can also load custom files, select monthly or weekly data, include pre/post-market hours, and export the data. Performance information can be displayed as well.

Examples:
- Load a stock ticker for analysis: stocks/fa/load -t <SYMBOL>
- Load a stock ticker with a specific start and end date: stocks/fa/load -t <SYMBOL> -s 2020-01-01 -e 2022-12-31
- Load a stock ticker with intraday data at 30-minute intervals: stocks/fa/load -t <SYMBOL> -i 30
- Load a stock ticker with weekly data: stocks/fa/load -t <SYMBOL> --weekly
- Load a stock ticker with monthly data: stocks/fa/load -t <SYMBOL> --monthly
- Load a stock ticker and include pre/post-market hours: stocks/fa/load -t <SYMBOL> -p
- Load a custom file for analysis: stocks/fa/load -f FILEPATH
- Load a stock ticker from a specific data source: stocks/fa/load -t <SYMBOL> --source AlphaVantage
- Load a stock ticker and export data in xlsx format: stocks/fa/load -t <SYMBOL> --export xlsx
- Load a stock ticker and display performance information: stocks/fa/load -t <SYMBOL> --performance
