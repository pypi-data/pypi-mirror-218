Summary: Load a stock ticker for analysis with options to specify the date range, interval, data source, and more. You can also choose to include pre/after market hours, load custom files, or use monthly and weekly data. Additionally, you can export the data in various formats and display performance information.

Examples:
- Load stock data for <SYMBOL> with default settings: stocks/dps/load -t <SYMBOL>
- Load stock data for <SYMBOL> with a specific date range: stocks/dps/load -t <SYMBOL> -s 2020-01-01 -e 2022-12-31
- Load stock data for <SYMBOL> with 15-minute intervals: stocks/dps/load -t <SYMBOL> -i 15
- Load stock data for <SYMBOL> including pre/after market hours: stocks/dps/load -t <SYMBOL> -p
- Load stock data for <SYMBOL> from a custom file: stocks/dps/load -t <SYMBOL> -f /path/to/custom/file
- Load monthly stock data for <SYMBOL>: stocks/dps/load -t <SYMBOL> -m
- Load weekly stock data for <SYMBOL>: stocks/dps/load -t <SYMBOL> -w
- Load stock data for <SYMBOL> and display performance information: stocks/dps/load -t <SYMBOL> --performance
- Load stock data for <SYMBOL> and export data in csv format: stocks/dps/load -t <SYMBOL> --export csv
- Load stock data for <SYMBOL> and save data to a specific excel sheet: stocks/dps/load -t <SYMBOL> --sheet-name Sheet1
- Load stock data for <SYMBOL> using a specific data source: stocks/dps/load -t <SYMBOL> --source AlphaVantage
